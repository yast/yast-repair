I was having the same problem on a Dec Alpha. I couldn't figure it out
until I di an ls -ls on my home dir. The netscape disk cache directoy
was almost 7Mb. In Netscape it was only set for 5MB. Well I had
Netscape clean the disk cache out, and I reduced the size to
500Kb. Occasionally I still have to clean out the disk cache as it
still seems to get bigger then it is supposed to (I do use the newest
netscape). So try doing this and see what happens it worked for me.:-)

 I think the clean disk cache is in preferences

: I *hate* background images, they are usually a waste of bandwidth
: and make the pages harder to read. So told netscape to ignore them. 
: To do this, just put the following line in your .Xdefaults
: (or whereever you store your X resources): 
: Netscape*documentColorsHavePriority: False
