Configuration

These articles refer to problems with the configuration of a Linux system. The configuration of the Linux kernel is the topic of the kernel-section. Some changes in the configuration can occur in a corrupted Linux system.

List of articles

General configuration problems

  1. System hangs at bootup after configuration of /etc/rc.config.d/rc.local:
    The system can't boot after some changes in /etc/rc.config. The system has to be restarted with a bootable disk and the file must be reset.
    Article in comp.os.linux.alpha or local.
    Research procedure: repair OR rescue
