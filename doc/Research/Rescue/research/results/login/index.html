General login problems

  1. LOGIN.GEN.1
    Login impossible:
    Use a rescue disk to boot and login. The partitions have to be mounted by hand.
    References:
    Linux-FAQ or local,
    comp.os.linux.* or local.

  2. LOGIN.GEN.2
    Forgot root password:
    The system has to be booted with a rescue disk. Mount the root partition by hand and edit the password files manually.
