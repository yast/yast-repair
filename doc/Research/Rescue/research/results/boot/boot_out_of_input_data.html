Beim Bootvorgang: ran out of input data

Cesky

Supportdatenbank (inputdata)

Symptom:

Beim Booten bricht der Kernel mit folgender Meldung ab: 
LILO Load Linux...............
Uncompressing Linux...

ran out of input data

-- System halted

Ursache:

LILO versucht einen Kernel zu laden, der nicht mehr vorhanden ist, weil mittlerweile ein neuer installiert wurde.

Lösung:

Booten Sie das System über den linuxrc Menüpunkt "Installiertes System booten", den Sie beim Start der Installation mit YaST1 angeboten bekommen. Beschrieben ist das z.B. im Handbuch von SuSE Linux 6.4, Kapitel 2.2.3 "Die Grundlegung: linuxrc".

Führen Sie dann einfach noch einmal LILO aus bzw. installieren Sie ihn mit YaST noch einmal. Beschrieben ist das im Handbuch Kapitel 3.6.2 "Kernel- und Bootkonfiguration" (SuSE Linux 6.4).
