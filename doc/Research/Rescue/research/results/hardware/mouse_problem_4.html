Normally you would edit your XF86Config file (somewhere in the "/etc"
hierarchy) and add a line like "ZAxisMapping 4 5" in the Pointer section.
 You might also need to change your pointer protocol to "IMPS/2" for
wheel support.  The current XFree86 config files are well commented so you
should be able to find it with no problem.  The XFree86 HOWTO should also
have all this information, but it's a bit intimidating for the uninitiated.

One drawback, though.  An application has to be written so that it
specifically provides mouse wheel support.  For instance, on my RedHat 7.0
system Mozilla 0.8.1 does mousewheel scrolling just fine, but the version
of Netscape (4.75) that RedHat includes with the 7.0 distro doesn't grok
wheelmice at all. Fortunately most of the rest of the Gnome and KDE stuff
works as it should.

I've got a Logitech cordless wheelmouse, and it's an absolute delight to
use.  Good luck.

There's a complicated clicky-pointy way I am sure, but all I did was add 
the line 

ZAxisMapping    4 5

to the XInput section of the /etc/XF86Config file, this is for a usb 
cordless mouse, so you might need to put that line somewhere else like the 
Pointer section if you are not using usb.
