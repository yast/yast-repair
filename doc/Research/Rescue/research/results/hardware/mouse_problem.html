You're using the wrong drivers. Well, in X Windows, for example, I use
a Microsoft compatible mouse (made by Microsoft), but when I told it
that I'm using an Intellipoint (which was NOT what the autoprobe
said), then it wouldn't leave, for me, the upper right hand corner.

To solve this, at the prompt, just type 'XF86Setup' and reconfigure
your mouse. Or, you can, at the prompt, type 'mouseconfig'. Either
one. I know XF86Setup will set it for X Windows, but mouseconfig does
it for all around, I think.
