Ok, you typed the root password for maintenance.  Did you do any maintenance,
or did you just type ^D?  

For maintenance, what you want to do is to run fsck against the file system
with the problem:

        # fsck -y /dev/hda5
        
When it's done, type ^D.  Yeah, there will many errors with "read-only file
system", but that's normal.  It'll then reboot and life will be wonderful.

        Stu
