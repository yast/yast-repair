> asap, as I did not delete win95, so it does have its uses.

If you know the size, start and end, and file system type of your linux
partitions before the debacle, then yes.

A couple of years ago, I blew my partition table trying to canabilize a
windoze extended partition for linux's use.

Luckily, I had my original partition table written on the back of an
envelope in pencil.

I booted to linux off an install disk (RH) and fdisk'ed my partition
table back together the way it was, and viola...

Keeping a copy of your "good" partition table around is always handy.
