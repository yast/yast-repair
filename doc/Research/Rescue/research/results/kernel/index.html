  2. KERNEL.GEN.2
    Error message: modprobe: can't locate module:

    References:
    Linux-FAQ or local,
    comp.os.linux.* or local,

  3. KERNEL.GEN.3
    PCMCIA not working after kernel update:
    The PCMCIA modules have to be upgraded first. The new kernel doesn't work with the old modules.
    References:
    Linux-FAQ or local,
    SuSE Support Database or local.

  4. KERNEL.GEN.4
    System map doesn't match actual kernel:
    Copy the file /usr/src/linux/System.map from /usr/src/linux to /.
    References:
    SuSE Support Database or local.

  5. KERNEL.GEN.5
    No SCSI support after kernel upgrade:
    The drivers for the root device have to be installed into the kernel, not as module.
    References:
    comp.os.linux.setup or local.

  7. KERNEL.GEN.7
    Kerrnel too big:
    Don't install too many modules directly into the kernel.
    References:
    SuSE Support Database or local.
