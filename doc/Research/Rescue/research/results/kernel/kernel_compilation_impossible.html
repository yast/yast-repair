Support knowledgebase (maddin_encaps)
Applies to

SuSE Linux: Versions since 5.0

Symptom:

The compiler aborts during compilation of a new kernel.

Cause:

It could be that there still is an old copy of /usr/bin/encaps installed. It was needed for compiling older kernels, but is not needed with current ones.

During compilation the system detects the installed binutils version by checking for /usr/bin/encaps.

Solution:

Newer versions of S.u.S.E. Linux should not have this program installed. Therefore you can simply remove /usr/bin/encaps.

This will not affect operation of the system, as no other part depends on it.
