Change of the file /etc/inetd.conf manually

Deutsch

Support knowledgebase (maddin_inet)
Applicable to
S.u.S.E. version: all
Kernel version: all

Symptom:

You have made changes manually in the file inetd.conf. Now, some network services do not run properly anymore.

Cause:

Possibly you have left leading blank spaces at the beginning of the line from the comment sign's removal. The inetd does not allow that.

Solution:

Please remove in the file inetd.conf every leading blank space of a line.
