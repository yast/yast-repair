what kind of a network are you connected to?
If you run ifconfig after a boot and see eth0 listed, than the card is working, 
but you can't get on the network.  
Do you have a static ip?  dhcp?  pppoe?

If you don't see eth0 with ifconfig than you probably need to mess with
/etc/conf.modules (or is it /etc/modules.conf?) and make sure the right module is being loaded
at boot.

With RedHat I suggest using Linuxconf first.
